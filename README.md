## COVID-19 Tracker

### Brief overview

- [ ✔ ] Fetches COVID-19 data through https://covid19.mathdro.id/api.
- [ ✔ ] Displays confirmed, recovered, and deaths number by country.
- [ ✔ ] Bar charts show the tendency.

### Language and Tools
- JavaScript, React, node.js, Hooks, Material UI, Charts js
